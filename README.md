# AlphaEarth Embeddings
This repository explores aggregation approaches for ports using Google's AlphaEarth embeddings.

To reproduce the results in this repository, create a virtual environment (recommended) and run
```
pip install -r requirements.txt
```
after activating the virtual environment.

## Notes
- `approach.md` briefly documents the steps taken to aggregate embeddings.
- Code present in `notebooks/baseline_creation.ipynb`.
